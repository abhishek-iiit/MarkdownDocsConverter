# Markdown Input / Output

A simple Next.js app for live Markdown editing and preview.

## Features

- Paste or type Markdown in the input area
- See rendered Markdown output instantly
- Supports GFM via `remark-gfm` (tables, task lists, strikethrough, etc.)

## Tech stack

- Next.js (App Router, TypeScript)
- `react-markdown`
- `remark-gfm`

## Local setup

1. Install dependencies:

```bash
npm install
```

2. Run development server:

```bash
npm run dev
```

Open `http://localhost:3000`.
